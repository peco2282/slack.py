Welcome to slack.py's documentation!
====================================

slack.py is a modern, easy to use, feature-rich, and async ready API wrapper
for slack.

**Features:**

- Modern Pythonic API using ``async``\/``await`` syntax
-  API and interactive components of the platform  by utilizing websockets.
